Краткое описание

В ходе тестирования был применен метод 30 тестов, который позволяет оценить работу сайта. Суть тестирования заключается в проверке работы всех обязательных функций сайта и его корректное отображение во всех актуальных браузерах.

Проверка web сайта выполняется с помощью актуальных браузеров IE, Edge, Chrome, Firefox, Safari, Opera; выявление возможных ошибок выполняется с помощью приложения VScode.
